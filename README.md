# Project Name
A Smart App-Based Communication and Emergency Assistance Tool for Deaf/Mute Users

---

## Folder Structure
mini-project/
├── frontend

└── backend

---
## Prerequisites

- Node.js and npm installed
- Expo CLI installed globally (`npm install -g expo-cli`)
- MongoDB URL already set in `.env` file
- **Expo Go** installed on your mobile device (from Play Store or App Store)

---

## Installation & Running the Project

You can run both frontend and backend as follows:

### Backend

```bash
cd backend
npm install
npm run dev

### Frontend

```bash
cd frontend
npm install
npx expo start
